# Дашборд для анализа продаж медицинских пробирок

**Цель проекта:** Создать автоматизированный инструмент для мониторинга ключевых метрик продаж медицинских пробирок, чтобы предоставить менеджменту возможность оперативно контролировать бизнес-показатели и выявлять точки роста.

**Задачи:**
*   Визуализировать общие метрики бизнеса: выручка, количество продаж, средний чек.
*   Проанализировать структуру продаж по ключевым срезам: **продукты, категории, регионы, поставщики, тип клиента**.
*   Определить ключевые драйверы выручки и самые популярные товарные позиции.
*   Обеспечить интерактивность для самостоятельного исследования данных менеджерами.

**Стек технологий:** `Yandex DataLens`.
---

## Ключевые выводы и бизнес-инсайты

1.  **Масштаб бизнеса:** Общая выручка за период составила **325 млрд ₽** (40 млн проданных единиц).
2.  **Товарный хит-парад:** Топ-3 продукта по количеству продаж дают почти идентичный вклад (~4 млн шт. каждый). Это **пробирки для анализа крови, хранения и общие для образцов** — ядро продуктовой линейки.
3.  **География продаж:** Традиционные лидеры — **Москва и Санкт-Петербург**. **Краснодарский край** показывает высокую активность, что может указывать на развитую сеть клиник в регионе.
4.  **Структура бизнеса:** Основной объем закупок (**~80%**) идет через ключевого поставщика **«МедТех»**, однако конечными покупателями чаще всего являются **частные лица**. Это говорит о B2C- или мелкооптовой модели продаж.
5.  **Драйверы выручки:** Категория **«Научные»** пробирки приносит наибольшую выручку (**>65 млрд ₽**), в то время как **«Клинические»** — лидер по объему продаж в штуках (**8 млн шт.**), что указывает на разницу в ценовом позиционировании.

##  Описание дашборда


Дашборд спроектирован как двухуровневый аналитический инструмент:

**Страница 1: «Обзор продаж» **
*   **KPI-панель:** Выручка, кол-во продаж, средняя цена.
*   **Динамика**: Динамика общей суммы продаж, средней стоимости товара.
*  **Продуктовый анализ:** Рейтинг продуктов по продажам, график взаимосвязи между объемом и суммой продаж.
*  **Структура продаж поставщиков клиентам**
*   **География:** Интерактивная карта России с интенсивностью продаж по регионам.
*   **Интерактивные фильтры:** Позволяют менять анализируемый период, выбирать конкретный регион, продукт, тип продукта, поставщика, тип покупателя или канал продаж для детализации.

**Страница 2: «Детализация по категориям»**
*   **Динамика суммы продаж по категориям:** Линейчатый график с отображением **процента прироста/снижения** для каждой категории (Научные, Клинические и др.).
*   **Динамика количества продаж по категориям:** Аналогичный линейчатый график для объема в штуках с **метрикой прироста**.
*   **Общая информация о проставщиках и клиентах**: Сводная таблица с данными о поставщиках и их клиентах.
*   **Интерактивные фильтры:** Возможность выбрать **отчетный период** и **период для сравнения** (например, «Апрель 2024 vs Апрель 2023»): это позволяет проводить YoY (год к году) и MoM (месяц к месяцу) анализ; выбор региона, поставщика, товара.

## Рекомендации для бизнеса

1.  **Управление ассортиментом:**
    *   **Стабилизировать** поставки топ-3 продуктов (пробирки для анализа крови, хранения, образцов), так как они формируют стабильный поток продаж.
    *   **Изучить потенциал** категории «Научные» пробирки: почему при высокой выручке объем в штуках ниже? Возможно, это премиум-сегмент с высокой маржой.

2.  **Развитие продаж:**
    *   **Тип клиента:** Преобладание частных лиц требует развития онлайн-каналов продаж и программ лояльности. Необходимо **углубить сегментацию** этой группы.
    *   **География:** Провести **отдельный анализ** по регионам-аутсайдерам для выявления барьеров (логистика, конкуренция, отсутствие представителей).
    *   **Зависимость от поставщика:** Диверсифицировать цепочку поставок, чтобы снизить риски, связанные с зависимостью от «МедТеха».

3.  **Мониторинг:**
    *   **Внедрить регулярный (еженедельный/ежемесячный)** просмотр дашборда для отслеживания динамики ключевых метрик.
    *   Настроить **алерт** на резкое падение продаж по топовым категориям или в ключевых регионах.
